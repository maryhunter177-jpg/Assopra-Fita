
# Sopra Fitas — Documentação Oficial

## Visão Geral
Sopra Fitas é uma aplicação web em React + Vite com temática retrô focada em jogos clássicos,
emulação e experiência nostálgica.

## Stack
- React + Vite
- JavaScript
- Supabase
- Vercel

## Estrutura
public/: assets públicos, capas e roms  
src/: código fonte React  
components/: componentes reutilizáveis  
pages/: páginas principais  
constants/: dados fixos  

## Funcionalidades
- Listagem de jogos
- Emulação
- Login e perfil
- Ranking
- Área administrativa

## Execução
npm install
npm run dev
