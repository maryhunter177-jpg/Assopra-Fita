# Arquitetura Geral

## Visão Geral
O projeto Sopra Fitas é uma aplicação web frontend construída em React com Vite,
integrada ao Supabase para autenticação e dados.

## Fluxo de Dados
Usuário -> Interface React -> Supabase -> Interface React

## Separação de Camadas
- Pages: telas principais
- Components: componentes reutilizáveis
- Constants: dados fixos
- Public: assets e ROMs
